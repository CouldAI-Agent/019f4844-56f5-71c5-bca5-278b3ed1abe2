# Project Proposal Format Generator

A Flutter application to quickly generate and manage project proposals.

## Features
- Dashboard to view existing project proposals.
- Structured form to create and edit proposals with standard fields (Overview, Objectives, Scope, Timeline, Budget).
- Detailed view to review proposal content.
- Clean and modern Material 3 design.

## Technical Stack
- Flutter (Cross-platform support for iOS, Android, Web, and Desktop)
- Dart
- uuid (for generating unique proposal identifiers)

## Setup
1. Clone the repository
2. Run `flutter pub get`
3. Run the app with `flutter run`
